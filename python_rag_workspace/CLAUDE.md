# python_rag_workspace

Python / RAG(검색 증강 생성) 프로젝트 전용 워크스페이스.
이 폴더 하위의 모든 Python 프로젝트는 아래 **공통 규칙**을 상속한다.
프로젝트 고유 규칙·설정은 각 프로젝트 폴더의 `.claude/` 에 따로 둔다.

## 공통 규칙 (`.claude/rules/`)

- `python-conventions.md` — 구조·명명·타이핑·도구·RAG 레이어 분리 규칙 (스타터)

> 이 워크스페이스는 아직 실제 프로젝트가 없는 **스타터** 상태다.
> 첫 프로젝트를 붙일 때 `python-conventions.md` 를 실제 스택에 맞게 다듬는다.

## 강제 레이어 (이 워크스페이스 밖)

push/시크릿 차단 같은 **강제 훅은 전역 `~/.claude/`** 에서 동작한다.
상위 워크스페이스 폴더의 `settings.json` 은 하위 프로젝트로 상속되지 않으므로,
이 워크스페이스는 규칙 문서(`.md`)만 담는다.
